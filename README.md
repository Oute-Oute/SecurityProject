# tp-securite

## Project setup
Clone project : ``https://github.com/Oute-Oute/SecurityProject``
then run : 
```
npm install
```
to install all dependencies.

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Project Deployed on Netlify :
